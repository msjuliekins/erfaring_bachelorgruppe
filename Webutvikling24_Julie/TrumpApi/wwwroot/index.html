About

This API contains information about Donalt Trump and his merch

Endpoints

Add new merch

/register-merch

See all merch

http://localhost:5033/api/TrumpMerch

See merch by id

http://localhost:5033/api/TrumpMerch/{id}

Search, Update and Delete

/merch-search

Info About properties

Id: int

Title: string

Image: file

Size: string

Colour: string
