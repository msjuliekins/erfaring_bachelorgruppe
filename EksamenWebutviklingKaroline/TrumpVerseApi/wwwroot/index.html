HTTP Calls

The calls supported in this web api:

Get, Post, Put, Delete

Attributes

  • thoughts: string -- The URL root for accessing Thought resources
  • merch: string -- The URL root for accessing Merch resources
  • staff: string -- The URL root for accessing Staff resources

Thoughts

Get all thoughts

You can access the list of thoughts by using the api/thought endpoint

Get a single thought

You can get a single thought by adding the id as a parameter: /thought/4

Merch

Get all merch

You can access the list of Merch by using the api/merch endpoint

Get a single merch

You can get a single thought by adding the id as a parameter: /merch/4

Staff

Get all staff

You can access the list of Staff by using the api/staff endpoint

Get a single staff member

You can get a single staff member by adding the id as a parameter: /staff/4
